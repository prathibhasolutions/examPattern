# MockTest Platform

A clean, production-ready Django project skeleton for a mock test platform. This repository provides only the project structure and essential configuration, without business logic.

## Apps
- accounts
- testseries
- questions
- attempts
- evaluation
- reports

## Project Structure
- Global templates directory with `base.html`
- Static folders for CSS and JS under `static/`
- Media configuration for user uploads

## Notes
- No models, views, or business logic are included.
- Start implementing features within the respective apps as needed.
